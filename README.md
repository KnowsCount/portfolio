### 📥 Install | 安装

1. clone repo | clone 项目

```
git clone https://github.com/KnowsCount/portfolio
```

2. npm install | npm 安装

```
npm i
//or
cnpm i //in 🇨🇳China if you'd like
```

### 📜 Scripts | 脚本

1. 🧪 Dev

```
npm run dev
```

2. 🛠 Build

```
npm run build
```

The build should be stored in `./docs`
